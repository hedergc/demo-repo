# FinOps AI Cost Optimization Prototype

[Next.js 15 + App Router] with Tailwind CSS.

## Setup

### Requirements
- Node.js (18 or amound)
- next 15.0.0
- yarn (installed)

### Install and Run

```sh
ndp install 
next dev
```

### Scripts
- **dev** : next dev
- **build** : next build
- **start** : next start
